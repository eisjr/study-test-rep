

## Windows-приложения (Desktop applications)

&nbsp;- - - - - - - - - - - 

* **Определение**



Приложения, которые устанавливаются и запускаются в ОС Windows и используют её ресурсы напрямую.



* **Методы программирования**



Объектно-ориентированное программирование (ООП)

Окна, кнопки, формы — объекты

Событийно-ориентированное

Реакция на действия пользователя (клик, ввод)

Компонентно-ориентированное

Использование готовых элементов интерфейса



Архитектурные шаблоны

MVC

MVVM (очень распространён в WPF)



* **Технологии и языки**



C# — WinForms, WPF, UWP

C++ — MFC, Qt

Java — JavaFX

Python — PyQt, Tkinter





## Web-приложения

 - - - - - - - - - - -

* **Определение**



Приложения, которые работают через браузер и не требуют установки.



* **Методы программирования**



Клиент-серверное программирование

Клиент (браузер) ↔ сервер

Событийно-ориентированное

События DOM (click, submit)

Компонентный подход



Интерфейс из независимых компонентов

MVC / SPA-архитектура

MVC (классические сайты)

SPA (React, Angular, Vue)



* Технологии и языки



**Клиентская часть:**

HTML, CSS

JavaScript (React, Vue, Angular)



**Серверная часть:**

JavaScript (Node.js)

Python (Django, Flask)

PHP, Java, C#
