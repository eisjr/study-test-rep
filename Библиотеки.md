

# Библиотеки



* Определение



Библиотека — это набор готовых функций, классов и модулей, которые разработчик подключает к программе, чтобы не писать всё с нуля.



* Зачем нужны библиотеки



Ускоряют разработку

Уменьшают количество ошибок

Повышают читаемость кода

Позволяют использовать проверенные решения
